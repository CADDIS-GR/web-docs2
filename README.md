# 🎣 CADDIS-GR: 플라이 낚시 타잉 도감

이곳은 **플라이 낚시의 타잉(Tying) 재료**들을 체계적으로 정리하고 소개하는 공간입니다. 
관리형 저수지에서 사용하는 다양한 패턴과 그에 필요한 재료 정보를 기록합니다.

---

## 🛠 오늘의 타잉 재료 소개

### ✨ TMC Prismatique Thread (TMC 프리즈마틱 스레드)
* **특징:** 티엠코(TMC)에서 출시한 특수 스레드로, 빛의 각도에 따라 영롱하게 반짝이는 효과가 있어 물고기의 유혹을 극대화합니다.
* **용도:** 미지(Midge) 패턴의 몸통이나 포인트 컬러를 줄 때 아주 유용합니다.
